# Trackerrr

A Vehicle Location Tracking System built using arduino uno, gsm module and gps module. An android app to show location on map is also developed.

## Connections:
### GPS module -- Arduino
* GND -- GND
* VCC -- 3.3V
* RX -- 11(Digital Pin)
* TX -- 10(Digital Pin)

### GSM module -- Arduino
* GND -- GND
* VCC -- 5.0V
* RX -- TX
* TX -- RX
